Get the number of characters in a composed message
Patch is available for version: alpine 1.00 alpine 1.10 alpine 2.00 alpine 2.01 alpine 2.10 alpine 2.11 alpine 2.20 alpine 2.21
 
This patch gives you the ability to count the number of characters that you send in a message. This is useful if you are sending a text message to a mobile phone, which is getting too long and you want to make sure it will arrive complete to recipient.
 
In order to count the number of characters, simply press the Ctrl-C command. In the menu on the bottom, there is the new command T, which allows you to counT the characters and lines in the message.
